genOverlay

genOverlay(setter: any, setter.overlay.overZIndex: any, setter.overlay.overColor: any, setter.overlay.overOpacity: any): any
Parameters
setter (any)
setter.overlay.overZIndex (any) : ha un valore di default altissimo nel caso è settabile
setter.overlay.overColor (any) : di default è bianco, indicare separati da virgola i valori rgb

Returns

genButton

genButton(setter: any, setter.button.text: any, setter.button.backColor: any, setter.button.cssStyle: any, setter.button.class: any)
Parameters
setter (any)
setter.button.text (any) testo da visualizzare sul bottone, default è "OK"
setter.button.backColor (any) colore del bottone default è "green"
setter.button.cssStyle (any) style inline del bottone nel formato css "prop:value;...prop:value;"
setter.button.class (any) , classi da assegnare al box formato stringa separata da spazi

init

function init le proprietà del setter sono tutte opzionali

init(setter: any, setter.overlay.overZIndex: any, setter.overlay.overColor: any, setter.overlay.overOpacity: any, setter.box.class: any, setter.box.cssStyle: any, setter.button.text: any, setter.button.backColor: any, setter.button.cssStyle: any, setter.button.class: any, setter.button.onClick: any, setter.text.title: any, setter.text.message: any, setter.title.class: any, setter.title.cssStyle: any, setter.message.class: any, setter.message.cssStyle: any): any
Parameters
setter (any) = oggetto di settaggio
setter.overlay.overZIndex (any) : (number) ha un valore di default altissimo nel caso è settabile
setter.overlay.overColor (any) : (string) di default è bianco, indicare separati da virgola i valori rgb
setter.overlay.overOpacity (any) : (0<number<1) default = 0.5
setter.box.class (any) : (string) elenco delle classi da applicare separate da uno spazio
setter.box.cssStyle (any) :(string) elenco delle regole css separate da ';' esempio: "padding: 0px; .... border: 1px solid red;"
setter.button.text (any) : (string) testo visualizzato nel bottone - default = 'OK'
setter.button.backColor (any) : (string/#code) colore del bottone default è "green"
setter.button.cssStyle (any) : (string) elenco delle regole css separate da ';' esempio: "padding: 0px; .... border: 1px solid red;"
setter.button.class (any) : (string) elenco delle classi da applicare separate da uno spazio
setter.button.onClick (any) : (function) funzione da eseguire su click del bottone ok, di default chiude la finestra
setter.text.title (any) :(string) titolo di xAlert
setter.text.message (any) : (string) testo da mostrare nel messaggio
setter.title.class (any) : (string) elenco delle classi da applicare separate da uno spazio
setter.title.cssStyle (any) :(string) elenco delle regole css separate da ';' esempio: "padding: 0px; .... border: 1px solid red;"
setter.message.class (any) : (string) elenco delle classi da applicare separate da uno spazio
setter.message.cssStyle (any) :(string) elenco delle regole css separate da ';' esempio: "padding: 0px; .... border: 1px solid red;"*
Returns
any: divElement, divElement ritorna un div che fa da overlay e un div che mostra il messaggio
